# Contributing to @compilr-dev/cli

Thanks for your interest in contributing to the compilr.dev CLI.

## Reporting Issues

Found a bug or have a feature idea? [Open an issue](https://github.com/compilr-dev/cli/issues/new/choose) using one of our templates:

- **Bug Report** -- Include your CLI version (`compilr --version`), the LLM provider you're using, and steps to reproduce.
- **Feature Request** -- Describe the problem you're solving and your proposed solution.

Good bug reports include:
- The exact command or action that triggered the issue
- What you expected vs. what happened
- Your environment (OS, Node.js version, terminal)
- Any error output from the terminal

## Source Code

This repository is a **public mirror** for issue tracking and documentation. The source code is developed in a private repository. We publish compiled packages to npm.

**Pull requests are not currently accepted.** If you have a code contribution in mind, please open an issue first to discuss the approach.
